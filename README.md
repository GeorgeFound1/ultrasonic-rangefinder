# Ультразвуковой дальномер (Университетский проект)

Инженерный проект автономного ультразвукового дальномера, разработанный в рамках обучения в НИЯУ МИФИ. Устройство представляет собой законченный физический прибор в корпусе с интерфейсом управления и энергонезависимой памятью для хранения замеров.

---

## 🛠️ Описание и ключевые фичи

* **Цифровая фильтрация данных:** Измерения расстояния защищены от случайных всплесков и шумов с помощью **алгоритма медианной фильтрации** по выборке из 3-х последовательных отсчетов.
* **Встроенный блок памяти (EEPROM):** Устройство поддерживает сохранение до 8 независимых слотов измерений. Данные сохраняются автоматически и не сбрасываются при отключении питания или смене аккумулятора.
* **Два режима измерения:** Прибор учитывает габариты собственного корпуса. С помощью физического переключателя можно переключать точку отсчета (от передней грани или от задней с автоматическим добавлением `+12 см` к замеру).
* **Кастомный GUI на LCD дисплее:** Для вывода текущего расстояния используется библиотека `BigFont01_I2C`, которая программно отрисовывает огромные, легко читаемые цифры на стандартном дисплее 20x4 за счет комбинации спецсимволов.
* **Продвинутый интерфейс (Энкодер):** Навигация и работа с памятью реализованы через один инкрементальный энкодер:
  * *Поворот:* Выбор ячейки памяти (1–8).
  * *Одиночный клик:* Вход в режим живого измерения / фиксация замера в выбранную ячейку.
  * *Удержание кнопки:* Сброс значения текущей ячейки памяти.
  * *Двойной клик:* Полная очистка всей базы данных EEPROM.

## 📂 Структура репозитория

* `/src` — исходный код прошивки (`rangefinder.ino` файл для Arduino).
* `/hardware` — принципиальная схема устройства в форматах PDF и PNG (спроектирована в draw.io).
* `/3d_models` — файлы корпуса для сборки и производства:
  * `rangefinder_assembly.step` — общая цветная 3D-сборка устройства (универсальный формат для CAD).
  * `case-front.stl` / `case-back.stl` — кастомные детали корпуса, готовые для отправки в слайсер и 3D-печати.

## 🔧 Зависимости и использованные библиотеки
Для компиляции проекта в среде Arduino IDE требуются следующие библиотеки:
1. `NewPing` — оптимизированная работа с ультразвуковыми датчиками через прерывания.
2. `EncButton` — продвинутый программный опрос энкодеров и кнопок с аппаратным антидребезгом и обработкой кликов/удержаний.
3. `LiquidCrystal_I2C` — управление дисплеем по шине I2C.
4. `BigFont01_I2C` — библиотека генерации больших шрифтов на символьных дисплеях.
5. `EEPROM` (Встроенная) — работа со встроенной энергонезависимой памятью микроконтроллера ATmega328p.
